# Pew Sheet upload — Admin guide

**For:** Fr Mark and church office  
**Updated:** June 2026

## What you do

Upload one file each week. The website updates automatically every **Saturday at 11:00am**.

## Where to upload

**Google Drive folder:** St Mary — Weekly Pew Sheet  
*(Karyna will share the link with you)*

Drag and drop your file into this folder. No website login needed.

## When to upload

- **By Friday night** — so it goes live Saturday morning
- If you upload on Thursday (as usual), that works perfectly

## File naming (required)

The filename **must include the word `pewsheet`** (lowercase).

```
pewsheet-2026-07-06.pdf
```

Use the **Sunday date** the pew sheet applies to.

Files without `pewsheet` in the name are **not found** by the automation (Drive search looks for that word).

## Accepted formats

- PDF (preferred)
- JPEG or PNG image

**Not accepted:** Word documents, multi-page booklets as ZIP, etc.

## What visitors see

- Preview on the **Community → Pew Sheets** page
- A **Download** button

Only the **latest** file is shown. Old files are replaced each week (not archived on the site).

## If something goes wrong

You will receive an email at **hello@stmaryrotherhithe.com** if:

- No file was found on Saturday morning
- The file type was wrong
- Google Drive could not be reached

The previous week's pew sheet stays live until a valid new file is published.
